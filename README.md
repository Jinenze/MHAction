这是一个简单的动作mod

动画api使用的是[KosmX](https://github.com/KosmX)的[PlayerAnimator](https://github.com/KosmX/minecraftPlayerAnimator)

感谢路上所有的mod制作教程作者

---
以下是一些计划

item

* 物品模型
* 物品第一第三人称渲染
* 伤害数值设计

gui

* 气刃值ui
* 锋利值ui
* 道具hud
client设置材质大小
* 道具gui
* 道具设置gui

action

* action动画
* 具体移动
* 攻击时机和碰撞箱大小

拔刀 踏步斩 收刀 翻滚

磨刀 见切 气刃斩 突刺

纳刀 居合 登龙 后跳斩

弹刀 击飞 飞扑 匍匐

数值

* 具体伤害计算
* 根据击中实体(部位？)计算伤害和锐利度以及是否弹刀
